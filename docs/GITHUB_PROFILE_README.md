## Top Projects

### [RTAI — Autonomous Red Team AI](https://github.com/CyberSentinel-sys/RTAI)
> Multi-agent penetration testing framework built on LangGraph + GPT-4o

```
ScoutAgent → AnalystAgent → StrategistAgent → FixerAgent
     ↓            ↓               ↓               ↓
 Stealth Nmap  CVE Database   3-Step LLM      Bash + Ansible
 ARP Sweep     Dynamic Risk   Reasoning       Safety Filter
               Scoring        Chain           + Approval Gate
```

- 9 specialised AI agents orchestrated by a LangGraph `StateGraph`
- CVE-grounded risk ratings with a Dynamic Risk Score formula
- Auto-generates copy-paste Bash patches, IPTables rules, Ansible playbooks
- Safety filter catches disruptive operations before human approval
- Streamlit CISO dashboard with Plotly charts and network graph
- **Stack:** Python · LangGraph · OpenAI · Tavily · Nmap · Streamlit · Pydantic

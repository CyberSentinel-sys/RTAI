> agent_roster

The Swarm

Four specialized agents operating in a coordinated pipeline — each with a single, focused objective.

01 Scout Agent

Performs ARP/ICMP host discovery and deep Nmap service enumeration. Returns a structured JSON inventory of every live host, open port, and detected service on the target network.

scapy nmap host discovery

02 Analyst Agent

Cross-references Scout data against a CVE database to compute a Dynamic Risk Score for every discovered service. Outputs a ranked list of entry points sorted by exploitability and impact.

CVE lookup risk scoring ranking

03 Strategist Agent

Consumes the Analyst's ranked entry points and LLM-generates a prioritized attack strategy with clear rationale. Maps attack vectors to MITRE ATT&CK techniques for structured reporting.

LLM planning MITRE ATT&CK prioritization

04 Fixer Agent

Reads the Strategist's plan and auto-generates actionable remediations — shell scripts, iptables rules, and Ansible playbooks — for every discovered vulnerability. Ready to deploy.

bash scripts iptables ansible
Scout → ↓ Analyst → ↓ Strategist → ↓ Fixer

> safety_protocols

Operational Safety

Autonomous does not mean uncontrolled. RTAI is designed around human oversight at every critical decision point.

Approval Bridge

Before any offensive action is executed, the swarm pauses and presents its full strategy to the operator. No exploit runs, no payload is sent, without explicit human sign-off.

  • ▸ Full strategy review before execution begins
  • ▸ Operator can approve, modify, or abort at any stage
  • ▸ All decisions and approvals are logged with timestamps
  • ▸ Immutable audit trail for compliance and review
Human-in-the-Loop

Maintenance Windows

Engagements are scoped to defined time windows, preventing runaway operations outside authorized hours. The swarm will not initiate or continue actions outside a configured schedule.

  • ▸ Configurable start/end windows per engagement
  • ▸ Automatic suspension outside authorized hours
  • ▸ Scope-limited to declared target ranges only
  • ▸ Emergency stop command halts all agent activity instantly
Time-Scoped Operations

RTAI is designed exclusively for authorized penetration testing and red team engagements. Use only on systems you own or have explicit written permission to test. Unauthorized use is illegal and unethical.
